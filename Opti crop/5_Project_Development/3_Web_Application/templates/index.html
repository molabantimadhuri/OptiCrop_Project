📖 About OptiCrop

OptiCrop is an AI-powered crop recommendation system developed to help farmers choose the most suitable crop based on soil nutrients and environmental conditions. Using Machine Learning, the system analyzes Nitrogen (N), Phosphorous (P), Potassium (K), Temperature, Humidity, pH, and Rainfall to provide accurate crop recommendations for sustainable agriculture.
